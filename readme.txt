Theme based on use of couple of plugins:
	* <a href="https://github.com/CherryFramework/cherry-shortcodes/">Cherry Shortcodes</a> - for generating and formatting content on static pages.
	* <a href="https://github.com/CherryFramework/cherry-shortcodes-templater/">Cherry Shortcodes Templater</a> - provides use of pre-formatted blocks in shortcodes output
	* <a href="https://github.com/CherryFramework/cherry-data-manager/">Cherry Data Manager</a> - host-to-host easy transfer of Wordpress Website
	* <a href="https://wordpress.org/plugins/types/">Toolset Types</a> - creating and managing post-types and taxonomies
